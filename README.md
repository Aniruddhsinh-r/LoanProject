# LoanProject
Neoloan: A responsive loan management web page built with HTML, Tailwind CSS, JavaScript, jQuery, and Alpine.js.

## 🚀 Live Demo
👉 https://neoloan.netlify.app/

## 🛠️ Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (ES6)
- jQuery
- Alpine.js

## ✨ Features
- Responsive modern UI
- Loan application validation form
- EMI calculator
- Clean layout with Tailwind CSS
- Deployed on Netlify

## 📌 Purpose
This project was created to showcase frontend development skills and UI design for internship and entry-level opportunities.
